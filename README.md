# Synology ACME
## 通过ACME协议自动更新群晖泛域名证书的脚本
## 使用方法：
1、打开群晖File Station，上传本项目所有文件到群晖的文件夹下，文件夹不要有中文和特殊符号，比如/volume1/docker/acme；
2、根据自己的域名信息，修改config；
3、控制面板，任务计划，创建任务，任务名称写acme，用户账号选root，计划选每月重复，任务设置输入：
/volume1/docker/acme/cert-up.sh update >> /volume1/docker/acme/log.txt 2>&1
4、在此任务上点右键，选择“运行”；
5、如果证书没到期，运行以后不会有任何提示；
6、如果当前证书即将到期（距离到期时间不足30天），运行脚本后耐心等待3-5分钟，等浏览器的页面出现刷新的提示的时候，就表示证书已经更新完成；
7、只需要把任务运行一次即可，以后证书到期前的1个月内，系统会自动的去申请证书。
