# Synology ACME
## 通过ACME协议自动更新群晖泛域名证书的脚本
## 使用方法：
1、点击右边Releases标签，下载本项目文件并解压出来；

2、打开群晖File Station，上传本项目所有文件到群晖的文件夹下，文件夹不要有中文和特殊符号，比如`/volume1/docker/acme`；

3、根据自己的域名信息，修改`config`这个文件；

4、控制面板，任务计划，创建任务，任务名称写acme，用户账号选root，开始日期选下个月1日，每月重复，在任务设置框中复制以下内容粘贴进去：

`/volume1/docker/acme/cert-up.sh update >> /volume1/docker/acme/log.txt 2>&1`

5、在此任务上点右键，选择“运行”；

6、如果当前证书即将到期（距离到期时间不足30天），运行脚本后耐心等待3-5分钟，脚本运行结束后，域名证书会自动延期90天；

7、只需要把任务运行一次即可，以后每个月1号系统会自动去申请证书。
